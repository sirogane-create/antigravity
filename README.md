# 音声作品納品用フォルダ振り分けツール

音声作品の納品用に、ファイル名や種類に基づいて自動的にフォルダを振り分け、ファイル名を整頓するスクリプトです。

## 主な機能

1. **自動フォルダ仕分け**:
   - ファイル名が「サンプル」で始まるファイル： `[プロジェクト名]_サンプル/` へ移動
   - その他のファイル：
     - ファイル名に「SEあり」を含む場合： `[プロジェクト名]_音声データ/[拡張子]/SEあり/` へ移動
     - ファイル名に「SEなし」を含む場合（または含まない場合）： `[プロジェクト名]_音声データ/[拡張子]/SEなし/` へ移動

2. **ファイル名の自動クリーンアップ**:
   - 仕分け後、ファイル名から「SEあり」「SEなし」の表記（およびその前後の `_` や `-`）を自動的に削除し、綺麗なファイル名にリネームします。

## 使い方

1. `音声作品納品用フォルダ振り分け.command` をダブルクリックして起動します。
2. フォルダ選択ダイアログが表示されるので、仕分けを行いたい対象のフォルダを選択します。
3. 自動的に仕分けとリネームが実行されます。
